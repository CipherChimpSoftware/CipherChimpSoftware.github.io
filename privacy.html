Privacy Policy

Cipher Chimp Software Privacy Policy

This policy describes how Cipher Chimp Software handles information for its website, beta program, and Visual Studio extensions.

Last updated: May 2026

CodeInsight Git

CodeInsight Git does not collect, store, transmit, or share personal information, source code, repository contents, or project files.

The extension does not require users to create an account. Git analysis is intended to run locally inside Visual Studio.

Comment Weaver

Comment Weaver does not collect, store, transmit, or share personal information, source code, repository contents, or project files.

The extension does not require users to create an account. Documentation analysis is intended to run locally inside Visual Studio.

Beta communication

If you email Cipher Chimp Software for beta access, support, privacy questions, or security reports, the information you send may be used to respond to your request, troubleshoot reported issues, and improve the product.

Do not send confidential source code, passwords, private keys, or sensitive production data by email.

Website

This website is hosted using GitHub Pages. Cipher Chimp Software does not currently use website accounts, payment processing, advertising trackers, or newsletter sign-ups on this site.

Future changes

If future versions add optional telemetry, cloud services, licensing services, payment processing, or account features, this policy will be updated before those features are used.
